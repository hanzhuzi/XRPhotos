# XRPhotos
##### 基于PhotosKit框架的照片选择器和照片浏览器
##### 跟系统的照片库一样流畅的体验
##### 已适配iOS11和iPhoneX
# 运行环境
##### Xcode 8.x/9.x
##### iOS 8.0 or latter
